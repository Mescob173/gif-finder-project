# Gif Finder

## Name
Manuela Escobar

## Overview
Gif Finder is a responsive web application that allows users to search for GIFs using the Giphy API. Users can type a keyword into the search bar, submit the form, and view GIF results directly on the page.

## How to Use
1. Open the website.
2. Type a keyword into the search bar.
3. Click the Search button.
4. GIF results will appear below the search form.

## Technologies Used
